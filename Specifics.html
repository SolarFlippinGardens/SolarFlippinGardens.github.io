SFG0001 - The Prototype Panel

A not so artistic view of the SFG0001 panel. Exciting stuff or what!

Open Source:

Hosted on GitHub https://github.com/SolarFlippinGardens/SFG0001 with all appropriate files including:

  • All firmware to be available in the repository
  • All design files (e.g svg/dxf cutting files, Openscad etc.)
  • All electronics files (e.g Eagle schematics and PCB layout)
  • Collected data to be available through open access on Tinamous.com

Features:

  • 3D Printer heated bed to keep it warm at night.
  • Solar panels in the hope of collecting a parsing photon.
  • Internet connected with a Particle Photon.
  • LED lightshow as this is clearly the more interesting part.
  • Current measurement so we can track power in/out of this beast.
  • 380mm diameter hexagon to so it will fit a 400mm acrylic sheet.
  • Hexagon shape to make it time consuming to fit and get tidy edges with lots of wasted materials during cutting.
  • Probably the only SFG panel that might ever see the light of day.
